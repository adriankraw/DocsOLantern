Keybindings

  • Blender hat zu viele Keybindings
  • ABER?! drück einfach die Leertaste .... oder F2 ... ?!
  • Die wichtigsten Tasten:
Taste Ergebnis
g bewege das ausgewählte Objekt
s skaliere das ausgewählte Objekt
a markiere alles
aa entmarkiere alles
r rotiere das ausgewählte Objekt
  • sehr wichtig! Viele Tasten und/oder Funktionen bieten nach dem ausführen weitere Möglichkeiten.
  • So kann oft X Y oder Z gedrückt werden, um ein Funktion an eine Achse zu binden
  • So kann aber auch SHIFT-X SHIFT-Y oder SHIFT-Z gedrückt werden, damit eine Achse ignoriert wird.
